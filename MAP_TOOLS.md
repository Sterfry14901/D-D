# Map tools — what works with Realms of Fate (and how)

The game now has **🖼️ Map image** on the World tab (DM only): export a PNG/JPG from
any tool below, upload it, then use **📍 Move pins** to place your cities on it.
Uploads are compressed automatically and synced to the whole table, saved with the
campaign, and survive server restarts.

## The winners (free)

- **Azgaar's Fantasy Map Generator** — azgaar.github.io/Fantasy-Map-Generator —
  **the best one on your list.** Free, open-source, runs in the browser, generates
  a stunning whole-world map in one click, endlessly customizable.
  *Workflow:* generate → Export → PNG → World tab → 🖼️ Map image → 📍 pin your cities.
- **hextml** (hextml.playest.net) — fast free hex maps; good for quick regional maps.
- **rollforfantasy.com** map/town/dungeon creators — free, simple; town maps make
  nice handouts (use the 🖼 Handout board), dungeon maps work as battle maps
  (DM battle-map upload already exists).

## Paid, still compatible

- **Worldographer / Hexographer** — polished desktop app; its "child maps" idea is
  covered in-game a different way: world map (World tab) → city pages → battle maps.
  Export any map as an image and upload it.
- **ProFantasy (Campaign Cartographer)** — powerful but pricey and a steep learning
  curve. Only worth it if you already own it. Exports images → upload.
- **RPG Maps Forge bundle** (Austin's recommendation) — pre-made battle map packs.
  Check the license, but map packs like this are normally sold exactly for
  personal VTT use: download → DM battle-map upload. Good value if you want
  pretty encounter maps without drawing.

## Worth knowing, different purpose

- **World Anvil** — a worldbuilding wiki (lore, timelines, articles), not a map
  drawing tool. Fine to use alongside; you'd upload its map images the same way.
  Their terms don't allow scraping their content into other apps, so keep it as
  your private notebook.
- **opengeofiction.net** — a community-built fictional *Earth-style* map project;
  fun, but not really usable for a fantasy campaign.
- **mapforge-software.com links page / reedsy blog** — link directories; Azgaar is
  the best thing they point to anyway.

## Licensing rule of thumb

Anything you **generate yourself** (Azgaar, hextml, rollforfantasy, Worldographer)
is yours to upload. Anything you **bought** (RPG Maps Forge, ProFantasy assets) is
almost always licensed for your own games — upload away, just don't redistribute
the files themselves. Don't upload maps ripped from published adventure books.
